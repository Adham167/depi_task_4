## Features

- Add new tasks with **title, category, and date**.
- Categorize tasks: Personal, Work, Health, Family, Learning.
- Track **pending** and **completed** tasks separately.
- Mark tasks as complete and see **strike-through effect**.
- Clean architecture using **Provider** for state management.
